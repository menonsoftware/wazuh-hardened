# SOC Demo Documentation

This folder contains step-by-step documentation for deploying and operating the SOC demo stack.

## Documents

- `docs/1.QUICKSTART.md`
  Fast 5-minute onboarding path for first deployment.

- `docs/2.DEPLOYMENT_GUIDE.md`
  End-to-end setup from host prerequisites to first successful login.

- `docs/3.OPERATIONS_RUNBOOK.md`
  Day-2 operations: start, stop, status checks, logs, backup, restore, and reset.

- `docs/4.TROUBLESHOOTING.md`
  Common failure modes and resolution steps.

- `docs/5.FLOW_AND_ARCHITECTURE_DIAGRAMS.md`
  Detailed architecture and operational flow diagrams.
